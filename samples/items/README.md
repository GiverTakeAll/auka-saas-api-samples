# アイテム連携 API 

アイテム連携APIへのつなぎ込みサンプルです。

環境変数に下記を設定して実行してください。

| 環境変数名        | 説明                                              |
| ----------------- | ------------------------------------------------- |
| LINE_ACCOUNT_UUID | ALL GRIT が発行する LINE アカウントの UUID です。 |
| ITEM_GROUP_UUID   | ALL GRIT が発行するアイテムグループの UUID です。 |
| AUTH_TOKEN        | ALL GRIT が発行する認証トークンです。             |
